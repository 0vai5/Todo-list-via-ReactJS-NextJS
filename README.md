# Todo List

This is a simple todo list app built with Next.js and React.

## Getting Started

To get started, clone the repo and install the dependencies:

```
git clone https://github.com/0vai5/Todo-list-via-ReactJS-NextJS
cd todo-list-app
npm install
```

Then, start the development server:

```
npm run dev
```

The app will be running at http://localhost:3000.

## Features

The app has the following features:

* Add tasks
* Delete tasks
* Edit tasks
* Mark tasks as complete

## Code Structure

The code is structured as follows:

* `app/layout.js`: The layout component for the app.
* `app/page.js`: The main page component for the app.
* `globals.css`: The global CSS file for the app.

## How to Use

To use the app, simply add tasks to the list. You can do this by clicking the "Add Task" button and entering a title and description for the task.

To delete a task, click the "Delete" button next to the task.

To edit a task, click the "Edit" button next to the task.

To mark a task as complete, click the checkbox next to the task.

## Conclusion

This is a simple todo list app that can be used to keep track of your tasks. It is easy to use and has a clean, modern design.
